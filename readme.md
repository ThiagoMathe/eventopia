# 🎫 Eventopia

O **Eventopia** é uma plataforma completa para criação, gerenciamento e venda de ingressos para eventos. O objetivo é conectar organizadores e participantes de forma fluida, segura e rápida.

---

## 🛠️ Tecnologias Principais

- **Frontend:** React + TypeScript + Tailwind CSS
- **Backend:** Node.js + NestJS + Prisma
- **Banco de Dados:** PostgreSQL (Docker)
- **Pagamentos:** (a definir)

---

## 🚀 Funcionalidades (MVP)

- [ ] **Autenticação:** Cadastro e login com JWT.
- [ ] **Gestão de Eventos:** CRUD completo de eventos para organizadores.
- [ ] **Venda de Ingressos:** Integração com gateway de pagamento.
- [ ] **Ingresso Digital:** Geração de QR Code para check-in.
- [ ] **Dashboard:** Visão geral de vendas e participantes para o organizador.

---

## 📦 Como rodar o projeto (Em breve)

1. Clone o repositório: `git clone https://github.com/seu-usuario/eventopia.git`
2. Instale as dependências: `npm install`
3. Suba os containers do Docker: `docker-compose up -d`
4. Inicie o servidor: `npm run dev`
